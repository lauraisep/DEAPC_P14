# DEAPC_P14
Trabalho DEAPC - Gestão de Projetos

O principal objetivo da aplicação é otimizar a gestão de projetos através de uma plataforma centralizada. O sistema permite reunir dados de projetos, equipas e tarefas no mesmo espaço, garantindo a atualização das listas de tarefas e a monitorização contínua do progresso e do estado das atividades.

Gestor do projeto (Dono):
- Perfil: Privilégios totais da administração nos projetos que gere.
- Função/Funcionalidades: Cria novos projetos, define prazos, cria e atribui tarefas, e gere a constituição das equipas.

Administrador (Técnico Informático):
- Perfil: Privilégios totais de administração do sistema e da base de ados.
- Função/Funcionalidades: Responsável pela manutenção técnica da aplicação, gestão de acessos, segurança dos dados e resolução de falhas no servidor.

Colaborador (Cliente):
- Perfil: Privilégios exclusivamente de leitura (sem permissão de edição).
- Função/Funcionalidades: Acompanha visualmente o progresso do projeto, consulta o estado das atividades e visualiza as listas de tarefas, sem poder alterar os dados.
